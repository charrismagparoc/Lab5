# IDRMS v3.0 — Integrated Disaster Risk Management System
### Barangay Kauswagan, Cagayan de Oro City

## Setup
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
npm run preview
```

## Deploy
Push to GitHub, then import to Vercel or Netlify.
- Build command: `npm run build`
- Output directory: `dist`
